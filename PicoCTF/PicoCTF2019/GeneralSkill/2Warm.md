## Notas adicionales

Se convierte el valor dado a binario.

Dividendo  Resto

	42/2 = 21    0

	21/2 = 10    1

	10/2 = 5     0

	5/2 = 2      1

	2/2 = 1      0

	1/2 = 0      1

Resulatado = 010101 inverso 101010
